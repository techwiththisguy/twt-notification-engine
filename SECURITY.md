# Security Policy

## Supported Versions
This project is currently in active development. Security patches are applied to the latest version on 'main'.

## Reporting a Vulnerability
If you discover a security issue, please report it privately.

**Contact:** techwiththisguy@gmail.com

Please include:
- A description of the issue
- Steps to reproduce
- Any proof-of-concept details (if available)

Do not open plublic issues for security vulnerabilities.
